# GOF (Gangs of Four) Design Patterns
---
This repository contains examples of all 23 design patterns of GOF.

## Creational Design Patterns
1. Abstract Factory
2. Builder Design
3. Factory Method
4. Prototype
5. Singleton

## Structural Design Patterns
1. Adapter
2. Bridge
3. Composite
4. Decorator
5. Facade
6. Proxy

## Behavioral Design Patterns
1. Chain of Responsibility
2. Command
3. Interpreter
4. Iterator
5. Mediator
6. Memento
7. Flyweight
8. Observer
9. State
10. Strategy
11. Template Method
12. Visitor
